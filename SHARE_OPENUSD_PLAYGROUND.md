# OpenUSD Playground Share Package

This package contains the built OpenUSD Playground app.

## Easiest Way To Share

1. Go to https://app.netlify.com/drop
2. Drag the unzipped `openusd-playground` folder onto the page.
3. Netlify will give you a public link you can share.

## What Is Inside

- `index.html`: the app entry page
- `assets/`: the app scripts and styles

This is a static website. It does not need a backend, login, database, or API key.

## Source

The app was built from the local React source files in this workspace:

- `openusd-playground-source/src/OpenUSDApp.tsx`
- `openusd-playground-source/src/components/OpenUSDCertificationCoach.tsx`
- `openusd-playground-source/src/data/openusdCertification.ts`
